YOLO——朴实无华的执行者

  • code2arter

不知道给YOLO系列一个“朴实无华”的标签会不会被人锤爆狗头，不过在我看来YOLO每次都是当下技术集大成者，没有拍案叫绝的骚操作，但是却告诉你站着巨人的肩膀上所能看到的一切。

YOLO v1

cvpr 2016。图像分类(image classification)问题很好用深度神经网络来建模的，如图1所以，几层CNN/FC之后接softmax即可。那怎么把这个思路扩展到目标检测领域呢？

Joseph Redmon 想法也很简单，我把图片划分成7x7大小，在每个格子中进行image classification即可。对了，还要顺带给出定位。故这样设计网络(图2)：

  1. 每个格子预测格子里面的物体是什么类别；
  2. 每个格子预测两个box的得分，这个box得分和类别得分相乘就是最后的得分。训练的时候选择与IoU最大的框进行训练，label就为IoU。预测的时候两个box都用上。
  3. offset (x, y, w, h)中x, y为物体中心到格子的距离，w, h为图片长宽的比例（YOLO需要resize到448x448） 故最后feature map为 S x S x (B * 5 + C),在VOC的实验中，即7 x 7 x (2 *5 + 20)

YOLO v2

cvpr 2017。论文风格我很喜欢，但是也集大成者的基本格式了。Better一章说明了所做的涨点改进： BN 谁用谁说好，+2% ap；

高分辨率预训练 从原来的224x224预训练改成448x448，+4% ap；

Anchor boxes 加了anchor box，就可以解决v1 每个grid在训练阶段只能给一个最大IoU的问题，提高了召回；

Anchor boxes聚类 决定大小；

直接预测x,y 作者觉得RPN那种 x = (pred * w_a) + x_a 的方式很飘，甚至可以满图飞，不如直接预测x（0~1）, yfeature map上的像素位置（如图3）。h和w依然是个anchor的相对关系决定；

多尺度训练 这次是全卷积网络了，上次有fc。

YOLO v3

2018。coco ap 33。 residual block加入，改进了darknet；

FPN结构，现在是3层特征预测啦，s32/s16/s8

多分类转多个二分类，softmax转sigmoid

目标检测中的单阶段与两阶段框架

April 20, 2020

如今基于深度学习的目标检测主要有两种框架，单阶段（one-stage）和两阶段（two-stage）。所谓one-stage，即在最后的feature map直接输出预测结果，所谓two-stage，即在第一阶段之后，那其输出的框重新在feature map上拿出特征进行refine。目前此两中框架大有殊途同归的趋势，今天来一探究竟吧。

目标检测中的分类与回归

April 18, 2020

由于任务的特殊性，在目标检测中存在两个子任务，一个是判断物体种类（是什么），另一个是给出物体位置（在哪里），这里我们就称为分类与回归。本文就来说说有哪些论文从这两个任务的联系与区别出发解决detection领域的问题。主要为采样和网络设计两个方向。
